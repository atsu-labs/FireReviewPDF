# 建築図面注釈・計測ソフトウェア 取扱説明書 (FireReviewPDF)

本ソフトウェアは、既存のPDF図面を背景として読み込み、その上に高精度な距離計測、面積計算、注釈（テキスト・図形・マーカー・凡例・円弧）の追加・編集を行うための先進的なデスクトップアプリケーションです。

---

## 1. 主な機能

### A. ファイル操作・プロジェクト管理
* **PDFインポート:** 図面PDFを背景として読み込みます（複数ページ対応、解像度 150 DPI）。
* **背景差し替え:** 既に描画した注釈データを維持したまま、背景のPDFファイルのみを更新します。
* **プロジェクト保存/読込:** 編集中のアノテーションデータ、ページごとのキャリブレーション（縮尺設定）、色名辞書などをJSON形式（`.json`）で保存し、後から完全に再編集できます。
* **PDF書き出し:** 背景図面と描画した注釈を高精度に合成した、新しいPDFを出力します。

### B. 計測・描画ツール
* **キャリブレーション（縮尺設定）:** 
  * **2点指定方式:** 図面上の既知の2点間をクリックし、実寸法を入力することで縮尺（1pxあたりのmm数）を定義。
  * **縮尺指定方式:** 「1/50」「1/100」などの縮尺比率を直接指定してキャリブレーション可能。
  * **ページ別管理:** キャリブレーションデータはページごとに個別に保持・管理できます。
* **距離計測 (直線/折れ線):** 直線または折れ線（ポリライン）を描画し、実寸法を自動計算してラベル表示します。
  * *Shiftキーを押しながらドラッグすることで、15度単位の角度スナップに対応。*
  * 折れ線ツールの終点には、矢印（Arrow）や丸（Circle）などの始点・終点マーカーを設定可能。
* **面積計測 (多角形):** 多角形で囲んだ領域の面積（m² または mm²）を自動計算します。
  * *右クリックで形状を確定します。*
* **円描画 (半径指定対応):** クリック＆ドラッグで円を描画するほか、キャリブレーション済みであればプロパティパネルで半径の実寸法（m または mm）を直接数値入力して描画することも可能。中心点にクロス（十字）等の中心マーカーを設定可能。
* **円弧 (Arc) ツール:** 中心点からドラッグして半径を決定し、指定した角度幅（スパン）の円弧を描画。中心マーカーや半径補助線の表示にも対応。
* **マーカー配置ツール:** 「スクエア（四角）」または「チェック（✓）」スタイルのマーカーを図面上にワンクリックで配置できます。
* **凡例 (Legend) 作成ツール:** ページ内に配置されたマーカーのスタイルと色をリアルタイムに自動集計し、図面上に移動可能な「凡例テーブル」として動的に生成・配置します。

### C. オブジェクト編集・カスタマイズ
* **選択・移動:** 配置済みのオブジェクトを選択し、ドラッグで位置を微調整できます。
* **ノード（頂点）編集機能:** 折れ線や多角形を選択してノード編集モードに入ると、個々の頂点をドラッグして形状を後から自由に変更できます（ダブルクリックで頂点削除、線分上でダブルクリックして頂点追加も可能）。
* **計算結果ラベルの個別移動:** 距離・面積・円などの測定結果ラベルを、オブジェクトとは独立してドラッグ移動し、最適な位置に配置できます。形状を編集して再計算が行われた場合でも、指定したラベルのオフセット位置は自動で維持されます。また、ホバー時に対応するオブジェクトとの関係を示すガイド線が表示されます。
* **テキスト注釈の拡張:** 任意の場所にテキストを追加できます。
  * テキストの「囲み枠（Border）」表示（色・太さ変更可）に対応。
  * テキストから対象物を指し示す「引き出し線（Leader Line）」を追加し、テキスト移動に追従させることが可能。
* **プロパティパネル (右側):** 選択したオブジェクトの属性をリアルタイムに編集可能です。
  * テキスト内容、フォント、文字サイズ（図面固定スケール）。
  * 線の色、幅、不透明度、塗りつぶし色、塗りつぶし不透明度。
  * 各種装飾（始点/終点/中心マーカー、引き出し線、囲み枠、円弧スパン角など）。
* **オブジェクトナビゲーター (左側):** ページ内のオブジェクト一覧がツリー表示され、選択・編集・削除・順序整理を効率的に行えます。
* **マーカー集計と色名編集:** 左側パネルでマーカーの集計（色・スタイル別）を確認できます。さらに、色ごとに「任意の名称（例：消火器、スプリンクラーなど）」をページ別に編集・定義でき、これが凡例テーブルにリアルタイム反映されます。

### D. 操作性・ビュー制御
* **ズーム制御:** ツールバーのコンボボックスからズーム倍率を直接選択・手入力できます。
* **物理DPIベースの実寸大ズーム:** ディスプレイの物理DPI情報を自動取得し、印刷時のサイズと画面上の表示寸法が一致する「実寸大表示（100%）」に対応。`Ctrl + 0` のショートカットキーで瞬時に切り替えられます。
* **用紙サイズ表示:** ステータスバーに、現在表示しているPDFページの図面サイズ（例：A3: 420.0 x 297.0 mm）をリアルタイムに自動判定して表示します。
* **mm/m 表示単位の切り替え:** キャリブレーションと連動した統合環境設定ダイアログから、表示単位を「m」と「mm」の間で自在に変更でき、すべてのアノテーション表示が自動で再計算・再フォーマットされます。

## 2. セットアップと実行方法

### 必要環境
* Python 3.10 以上
* 依存ライブラリ: `PySide6`, `PyMuPDF` (fitz), `qtawesome`

### インストール
```powershell
pip install PySide6 pymupdf qtawesome
```

### 実行
```powershell
python src/main.py
```

## 3. 基本的な操作フロー

1. **図面を開く:**
   ツールバーの「PDFを開く」ボタン（または `Ctrl + O`）から背景となるPDF図面を選択します。ステータスバーには自動的に用紙サイズ（例：`A3`）が表示されます。

2. **縮尺を設定する (キャリブレーション):**
   * 右上の「環境設定（歯車）」ボタンをクリックし、「単位」および「縮尺」を設定します。
   * **既知の比率を使う場合:** 縮尺比率（例：`1/50` や `1/100`）を設定して適用します。
   * **図面上の実寸から合わせる場合:** 「図面からキャリブレーション」ボタンを押し、図面上のスケールバーなどの既知の2点をクリックして、ポップアップに実寸法（m または mm）を入力します。
   * 右上の「スケール」インジケータが「スケール: 1/50」のように有効化されます。

3. **計測・描画を行う:**
   ツールバーから任意のツールを選択し、図面上に書き込みます。
   * **距離/折れ線:** クリックで頂点を追加し、ダブルクリックまたは他ツール選択で確定。`Shift` キーで角度スナップ。
   * **面積:** クリックで多角形を描き、右クリックで形状を確定。
   * **円/円弧:** ドラッグして描画。必要に応じて右側のプロパティパネルで半径の実寸を入力。
   * **テキスト:** クリックして文字を入力。プロパティパネルから「囲み枠」や「引き出し線」の有無を設定可能。
   * **マーカー:** ツールバーでマーカー（スクエア/チェック）と色を選択し、図面をクリックして配置。「連続配置」チェックを入れておくと、連続してマーカーをスタンプできます。

4. **凡例を配置・集計する:**
   * 図面上にいくつかマーカーを配置すると、凡例ツール（Legend）が有効になります。
   * 凡例ツールを選択し、図面上の配置したい場所をクリックすると、自動集計された美しい凡例テーブルが配置されます。
   * 左側のナビゲーターパネル下部の「マーカー集計」リストで、各色の名称部分（デフォルトはHexカラーコードや基本色名）をダブルクリックして「消火栓」などのカスタム名に変更すると、図面上の凡例テーブルの文字もリアルタイムに更新されます。

5. **微調整・編集する:**
   * 「選択・移動」モード（`A` キー）に切り替え、オブジェクトを選択します。
   * 右側のパネルで線の太さ、色、塗りつぶし、不透明度などを調整します。
   * 計算結果のテキストラベルの数字部分をドラッグして、見やすい位置にずらします。
   * 頂点位置を微調整したい場合は、左側のナビゲーターパネルまたは右側のプロパティパネルの「頂点編集」を有効にし、表示された紫色のハンドルをドラッグします。

6. **結果の保存と書き出し:**
   * 編集中の作業状態を保存する場合は「プロジェクト保存」を行い、JSONファイルを書き出します。
   * 成果物として印刷・共有用PDFを出力する場合は「PDF書き出し」を行います。注釈が高解像度で合成されたPDFが生成されます。

## 4. 技術仕様（開発者向け）

### アーキテクチャ
* **GUIフレームワーク:** PySide6 (Qt for Python)
  * `QGraphicsView` / `QGraphicsScene` を使用したスムーズなズーム・パン、高精度な座標系管理。
  * 各種描画モードは **Stateパターン**（`BaseCanvasTool` を継承する各種ツールクラス）によってイベント管理を分離。
* **PDF処理エンジン:** PyMuPDF (fitz)
  * 150 DPIでの鮮明なラスタライズレンダリングと用紙サイズ（ミリメートル寸法）の自動解析。
* **データ永続化仕様:** 
  * 座標系はPDFのピクセル座標（150 DPIベース）を基準に統一。
  * JSON保存時はページごとのスケールファクター、注釈の幾何情報、テキスト装飾、凡例位置、マーカー集計用のローカル色名データを完全保持。
